**About CrewAI**

CrewAI is a Python-based framework used to build autonomous AI agents that collaborate as a team (crew). Each agent has:

A defined role

A specific goal

Tools to perform tasks

These agents communicate, delegate tasks, and work together—similar to a real-world team.

## Features

Multi-agent collaboration system

Role-based AI agents (researcher, writer, planner, etc.)

Task delegation between agents

Sequential or structured execution of tasks

Intelligent decision-making using LLMs

Modular and scalable architecture

## Tech Stack

Python

CrewAI Framework

LLM APIs (Groq / Local models)

Jupyter Notebook / Python Scripts

## How It Works

Define multiple agents with specific roles

Assign tasks to each agent

Create a “crew” that includes all agents

Execute the workflow

Agents collaborate and produce final output
